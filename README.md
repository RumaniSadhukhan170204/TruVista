# TruVista  
### Cloud-Integrated VR-Based Smart Forensic Investigation System

## Overview
TruVista is a Cloud-Integrated VR-Based Smart Forensic Investigation System designed to modernize crime scene analysis using **IoT, Cloud Computing, and Virtual Reality**. The system enables investigators to digitally reconstruct crime scenes in a secure virtual environment, ensuring evidence integrity while improving investigation efficiency.

## Problem Statement
Traditional crime scene investigations are time-consuming, prone to human error, and risk evidence contamination. Physical access limitations and lack of remote collaboration further reduce investigation effectiveness.

## Proposed Solution
TruVista leverages:
- **IoT devices** to capture real-time crime scene data  
- **Cloud infrastructure** for secure storage, processing, and scalability  
- **VR technology** to generate immersive 3D crime scene reconstructions  

Investigators can explore, analyze, and collaborate remotely without disturbing the original scene.
